# Re-job

## Setting up *.env* file

        DB_HOST=localhost
        DB_USER=username
        DB_PASS=password
        DB_PORT=3306

## Compiling and running

1. Install [Yarn](https://github.com/yarnpkg/yarn/releases/download/v1.12.3/yarn-1.12.3.msi)
2. Set up the *.env* file
3. Run:

        yarn
        nodemon app.js

4. The web-app will run at **localhost:3000**
